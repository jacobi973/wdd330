What I learned

  • An object can incude functions that can be called when the object name is called. ie. user = {sayHi: function(){...}}

  • The keyword This allows for it to be used in different functions, it isn't bound.

  • In JavaScript this is “free”, its value is evaluated at call-time and does not depend on where the method was declared, but rather on what object is “before the dot”.


  • EXAMPLES 
                    
                    function makeUser() {
                        return {
                          name: "John",
                          ref: this
                        };
                      }
                      
                      let user = makeUser();
                      
                      alert( user.ref.name ); // What's the result? The result awould be an undefined name or error that won't be able to access the ref name.

                      The beginning alerts were examples as well.

  • You can use multiple strings when getting object names ie. superman["real" + " " + "name"]
